# AlertDialog のテスト

## 何をテストするか

`AlertDialog` はブラウザの `alert` の代替となるモーダルコンポーネント。
エラーメッセージ表示、閉じる操作、アクセシビリティ属性を検証する。

## なぜテストするか

エラー発生時の通知表示をテスト可能にするため。
`alert` は jsdom やテスト環境ではモーダルを表示せず、後続のロジックもテスト不能になる。

## テストケース

### 表示
- メッセージが正しく表示されること
- closeLabel のカスタマイズが効くこと

### 操作
- OK ボタンクリックで onClose が呼ばれること
- Escape キーで onClose が呼ばれること
- 背景クリックで onClose が呼ばれること

### アクセシビリティ
- role="alertdialog" が設定されていること
- aria-modal="true" が設定されていること
